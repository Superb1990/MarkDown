# 列表  
### 无序列表
无序列表使用星号（*）、加号（+）、或者减号（-）作为列表标记   这些标记后面要加一个空格，然后再填写内容  

**使用星号的**  
* apple
* orgin
* banana

**使用加号的**  
+ dark
+ yellow
+ red

**使用减号的**  
- up
- down
- left
- right

<mark>建议：使用 减号-</mark>  

### 有序列表
使用数字加`.`来表示  (数字+.+空格)
1. 测试1  
2. 测试2  
3. 测试3  

可以不用按顺序，markdown会自动修正数字顺序  
1. 测试1  
3. 侧2  
5. 测试9  

从指定数字开始  
7. 端1  
11. 端2  
12. 端3
